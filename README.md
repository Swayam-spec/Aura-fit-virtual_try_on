# Aura-Fit Virtual Try-On Platform

A high-end, responsive Virtual Try-On (VTO) web application built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Designed to blend a professional e-commerce utility with a luxury immersive experience.

## Features
- **Dual-Column Workspace**: Clean layout separating Asset Management and Live Canvas.
- **Glassmorphic UI**: Minimalist navigation and floating elements with blurred backgrounds.
- **Immersive Animations**: Uses Framer Motion for scroll-linked 3D transitions, staggered reveals, and magnetic buttons.
- **Real-Time Generation UI**: Morphing loader with smooth gradient states and responsive status updates. 

## Tech Stack
- Frontend: Next.js (App Router), React
- Styling: Tailwind CSS
- Animations: Framer Motion
- State Management: Zustand
- Icons: Lucide-react

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Start the development server with `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Deployment
This project is fully ready to be deployed on Vercel. Connect this repository to your Vercel account and deploy with zero required configuration.
